## Dependencies
### Virtual environment
```bash
python3 -m venv venv		# create a virtual environment for the project
source venv/bin/activate	# activate it

which python				# check if venv is activated
							# (it should display path to the venv)
```
### Pydantic 2.x
```bash
pip install "pydantic>=2"	# install pydantic to the venv
```
